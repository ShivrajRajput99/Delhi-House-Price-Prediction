# Delhi House Price Prediction using Machine Learning

## Project Overview

Developed a Machine Learning model to predict residential property prices in Delhi using housing features and market-related data.

## Objectives

* Analyze housing market trends
* Clean and preprocess real-world data
* Build predictive machine learning models
* Evaluate model performance

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* Jupyter Notebook

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training
6. Prediction and Evaluation

## Key Skills Demonstrated

* Data Analysis
* Data Visualization
* Machine Learning
* Problem Solving
* Python Programming
